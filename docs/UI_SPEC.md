## 二、设计规范（Design System）

### 2.1 颜色系统

```
主色（Primary）     #00B96B   → 运行中状态、主要按钮、开关ON、添加操作
危险色（Danger）    #FF4D4F   → 停止监控、删除、清空、已过期状态
警告色（Warning）   #FF7A00   → 测试操作按钮、优先级星级
信息色（Info）      #1677FF   → 蓝色链接、数量高亮、操作链接按钮
成功色（Success）   #52C41A   → 成功徽章、已完成状态
标签色（Tag-Orange）#FA8C16   → 关键词过滤"包含匹配"徽章
标签色（Tag-Red）   #FF4D4F   → 昵称排除"排除"徽章
背景（Page BG）     #F5F5F5   → 整体背景
面板背景（Panel）   #FFFFFF   → 内容卡片、表格背景
侧边栏背景          #FFFFFF   → 左侧导航面板
边框色（Border）    #E8E8E8   → 分割线、输入框边框
文字-主（Text-1）   #1A1A1A   → 标题、重要信息
文字-副（Text-2）   #595959   → 普通文本
文字-三（Text-3）   #8C8C8C   → 占位符、次要说明
状态在线            #52C41A   → 账号在线状态
状态离线            #8C8C8C   → 账号离线
```

### 2.2 字体规范

```
字体族：PingFang SC, Microsoft YaHei, -apple-system, sans-serif

层级        字号    行高    字重
─────────────────────────────────
页面标题     16px   24px   600
模块标题     14px   22px   600
表格表头     13px   20px   500
表格内容     13px   20px   400
说明文字     12px   18px   400
徽章文字     11px   16px   500
数字数据     13px   20px   600（数字使用等宽字体 tabular-nums）
```

### 2.3 布局尺寸

```
应用窗口最小尺寸：1200 × 700px
标题栏高度：         48px（Wails 自定义标题栏 + 状态栏合并）
左侧导航宽度：       88px（固定，图标+文字竖排）
内容区域内边距：     16px
面板间距：           12px
卡片圆角：           8px
输入框圆角：         6px
按钮圆角：           6px
小徽章圆角：         4px
```

### 2.4 通用组件规范

#### 2.4.1 状态徽章（Status Badge）
```
形态：圆角矩形，内边距 2px 8px，11px 字体

运行中：  绿色背景(#F6FFED)  + 绿色文字(#52C41A) + 绿色圆点（·）前缀
已停止：  灰色背景           + 灰色文字           + 灰色圆点
就绪：    灰色背景           + 灰色文字
在线：    绿色背景           + 绿色文字（"在线"）
已过期：  红色背景           + 红色文字
成功：    绿色背景           + 绿色文字（"成功"）
失败：    红色背景           + 红色文字（"失败"）
INFO：    蓝色背景(#E6F7FF)  + 蓝色文字(#1677FF)
WARN：    橙色背景(#FFF7E6)  + 橙色文字(#FA8C16)
ERROR：   红色背景(#FFF1F0)  + 红色文字(#FF4D4F)
```

#### 2.4.2 数字步进器（Stepper）
```
结构：[—] [数字输入框] [+]
尺寸：— 按钮 24px×24px，输入框 60px 宽，+ 按钮 24px×24px
整体高度：32px
边框：1px solid #E8E8E8，圆角 6px
后缀单位：写在步进器右侧（如"分钟"、"秒"、"条"、"个"）
```

#### 2.4.3 开关（Toggle Switch）
```
尺寸：宽40px × 高22px
ON：主色 #00B96B，白色圆形滑块
OFF：灰色 #D9D9D9，白色圆形滑块
过渡动画：200ms ease
```

#### 2.4.4 操作链接按钮（Text Button）
```
登录：蓝色 #1677FF，无背景，hover 下划线
提取：蓝色 #1677FF
日志：蓝色 #1677FF
测试：橙色 #FA8C16
删除：红色 #FF4D4F
各按钮之间用竖线 | 分隔，字号 13px
```

#### 2.4.5 优先级星级（Star Rating）
```
5颗星，直径8px，间距2px
填充色（已选）：#FAAD14（金色）
空心色（未选）：#D9D9D9
非交互展示态（只读）
```

#### 2.4.6 数量标记计数器
```
显示在标题后，形态：圆形/椭圆，背景 #FF4D4F（黑名单红点）或 #1677FF（蓝色数字）
黑名单红点：直径18px，白色数字，竖排于图标上方
评论流数量：绿色圆角矩形标签，如 "60"
```
