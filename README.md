## 📋 项目简介

**抖音截流私信获客系统**是一款为抖音营销人员设计的自动化获客工具（研发中）。目标集成**评论监控、视频自动采集、智能私信转化**三大核心功能：

- 🔍 **实时监控**：自动监控目标视频评论区，第一时间发现潜在客户，无监控视频数量限制
- 🎯 **精准采集**：根据关键词、发布时间、互动数据等多维度条件自动采集优质视频，自动添加最新符合条件视频到监控列表
- 💬 **智能私信**：多账号自动发送个性化私信，实现评论截流和视频获客
- 📊 **数据沉淀**：完整记录客户信息和沟通历史，构建私域流量池

### ✨ 核心优势

| 优势 | 说明 |
|------|------|
| ** 🚀 目标架构** | Go 语言后端 + Wails 框架，多线程并发处理（研发中）|

## 🌟 核心功能

### 1️⃣ 评论监控系统 - 实时发现潜在客户

#### 📊 实时监控
- **多视频并发监控**：同时监控多个视频的评论变化
- **毫秒级响应**：评论发布后立即捕获，抢占先机
- **评论详情抓取**：自动获取评论者昵称、UID、IP地址、点赞数等完整信息

#### 🎯 智能筛选
```
✅ 关键词筛选 - 仅保留包含指定关键词的评论
✅ 昵称过滤 - 排除指定用户的评论
✅ 地区过滤 - 仅保留指定地区的客户
✅ 拉黑机制 - 自动屏蔽恶意用户
```

#### 📢 多渠道通知
- **钉钉机器人推送**：实时将新评论推送到钉钉群
- **本地日志记录**：完整记录所有监控历史
- **内存缓存优化**：大容量评论数据自动清理

#### 🧹 自动清理
- **定时清理过期评论**：可设置1天、7天、15天等自动清理策略
- **CID去重机制**：防止重复抓取相同评论
- **智能状态同步**：自动处理API缓存延迟问题

---

### 2️⃣ 视频自动采集 - 精准定位目标客户

#### 🎯 多维度搜索
```
✅ 关键词搜索 - 支持多个关键词组合搜索
✅ 排序方式 - 综合排序 / 最多点赞 / 最新发布
✅ 发布时间 - 不限 / 一天内 / 一周内 / 半年内
✅ 视频时长 - 1分钟以下 / 1-5分钟 / 5分钟以上
```

#### ⚡ 智能采集
- **批量关键词管理**：支持批量导入和管理多个关键词
- **优先级设置**：为不同关键词设置不同采集优先级
- **并发采集引擎**：支持1-10个关键词同时采集
- **实时进度反馈**：前端实时显示采集进度和统计信息

#### 🧩 智能过滤
```
✅ 点赞数过滤 - 设置点赞数范围（如：100-10000）
✅ 评论数过滤 - 设置评论数范围（如：10-500）
✅ 发布时间过滤 - 仅采集指定小时内发布的视频
✅ 作者过滤 - 排除/仅包含指定作者
✅ 标题过滤 - 标题必须包含/不能包含指定关键词
```

#### 🔄 去重机制
- **本地视频库去重**：已添加到监控的视频不会重复采集
- **历史采集记录**：可选择"仅采集新视频"模式
- **采集统计报表**：记录每个关键词的采集效果

#### 📤 自动添加
- **一键添加监控**：采集到的视频可自动添加到监控列表
- **批量导入视频**：支持从文本文件批量导入视频链接
- **视频ID提取**：自动从抖音分享链接中提取视频ID

---

### 3️⃣ 私信智能转化 - 自动化客户转化

#### 👥 多账号管理
- **比特浏览器集成**：支持多个抖音账号同时在线
- **账号状态监控**：实时显示账号在线状态
- **账号信息提取**：一键提取账号昵称、UID等信息
- **证书自动缓存**：24小时内复用证书，提升发送成功率

#### 📝 智能发送
- **模板消息**：支持自定义私信模板，支持多条模板随机发送
- **账号轮询机制**：自动轮询可用账号，避免单账号限流
- **时间槽预占**：防止并发发送时重复选中同一账号
- **失败重试**：网络错误自动重试3次，提升发送成功率

#### 📊 发送控制
```
✅ 日限额控制 - 单账号日上限、全部账号日上限
✅ 发送间隔控制 - 最小发送间隔（秒）
✅ 并发线程控制 - 支持1-10个线程并发发送
✅ 发送频率限制 - 智能控制发送节奏
```

#### 📈 数据统计
- **发送日志记录**：完整记录每条私信的发送状态
- **任务队列管理**：待发送、发送中、成功、失败状态管理
- **已发送用户追踪**：防止重复发送
- **每日统计报表**：今日发送数、成功率等关键指标

#### 🔐 安全机制
- **证书本地存储**：Token、SDKCert、私钥等敏感信息加密存储
- **账号状态检测**：自动检测账号登录状态
- **失败账号隔离**：连续失败账号自动暂停使用

## 📚 配置说明

### 评论监控配置
- **自动清理时间**：设置评论自动清理的天数（0表示不清理）
- **拉黑阈值**：设置用户出现多少次后自动拉黑
- **钉钉Webhook**：配置钉钉机器人推送地址

### 视频采集配置
- **采集间隔**：设置自动采集的时间间隔（分钟）
- **最大并发**：设置同时采集的关键词数量（无数量限制）
- **自动添加**：采集到的视频是否自动添加到监控
- **仅采集新**：是否只采集之前未采集过的视频

### 私信配置
- **模板路径**：私信模板文件路径（每行一条模板）
- **最小间隔**：账号最小发送间隔（秒）
- **单账号日上限**：单个账号每天最多发送数量
- **总日上限**：所有账号每天最多发送数量
- **并发线程**：同时发送的线程数量（1-10）

## 🔧 常见问题

### Q1: 如何提取抖音账号的证书信息？
A: 在私信账号管理页面，点击"提取"按钮，系统会自动从比特浏览器中提取账号的证书信息并缓存24小时。

### Q2: 私信发送失败怎么办？
A: 系统支持自动重试机制，网络错误会自动重试3次。连续失败3次的账号会被暂停使用。

### Q3: 如何防止账号被限流？
A: 系统采用智能账号轮询、时间槽预占、发送频率控制等机制，有效防止账号限流。

### Q4: 数据保存在哪里？
A: 所有数据保存在本地JSON文件中，包括：
- `app_data.json` - 评论监控数据
- `dm_config.json` - 私信账号和配置
- `video_collection_config.json` - 视频采集配置
